# BestDiffTool / trpoKursach

Десктопное приложение на **C++17** и **Qt 6 (Widgets)** для сравнения двух текстов: ввод вручную или загрузка `.txt` (UTF-8), построчный diff и подсветка отличий внутри строки.

## Возможности

- Два поля «Текст 1» и «Текст 2» с нумерацией строк (`CodeEditor` на базе `QPlainTextEdit`).
- Загрузка файлов, режим результата в двух таблицах (`QTableWidget`): номер строки + текст.
- Сравнение **по индексу строки** \(строка *i* слева соответствует строке *i* справа\): `Same` / `Changed` / `Removed` / `Added`.
- Перед анализом длинные логические строки режутся по константе `kReflowMaxColumnChars` в `mainwindow.cpp` (только для расчёта diff, исходный текст в полях не меняется).
- Внутри изменённой строки — подсветка фрагментов по символьной НВП (`intraline_diff`), отрисовка через `DiffTextDelegate` и `QTextDocument`.
- Синхронизация вертикальной прокрутки между панелями.

## Сборка

Требования: **CMake ≥ 3.16**, компилятор с **C++17**, **Qt 6** с модулем **Widgets**.

```bash
cmake -S . -B build -DCMAKE_PREFIX_PATH="путь/к/Qt/6.x/msvc2019_64"   # или ваш kit
cmake --build build
```

Исполняемый файл задаётся именем проекта в `CMakeLists.txt` (по умолчанию `trpoKursach`).

## Запуск

Запустите собранный `.exe` из каталога `build` (или через IDE). Введите тексты или загрузите файлы, нажмите **Анализ**; **Редактировать тексты** возвращает к режиму ввода.

## Структура исходников

| Файл | Назначение |
|------|------------|
| `main.cpp` | Точка входа, стиль/палитра приложения |
| `mainwindow.cpp` / `mainwindow.h` | UI, загрузка файлов, анализ, заполнение таблиц |
| `code_editor.cpp` / `code_editor.h` | Редактор с полосой номеров строк |
| `diff_engine.cpp` / `diff_engine.h` | Построчное сравнение по индексу |
| `diff_types.h` | Типы результата diff |
| `text_io.cpp` / `text_io.h` | Чтение текстового файла в строки |
| `intraline_diff.cpp` / `intraline_diff.h` | Подсветка отличий внутри одной пары строк |
| `diff_text_delegate.cpp` / `diff_text_delegate.h` | Делегат ячейки таблицы для HTML-фрагмента |
| `CMakeLists.txt` | Сборка CMake |
